## File Preparation Checklist

### For Each Design:

- [ ] **File Format**: PNG with transparent background OR PDF vector
- [ ] **Resolution**: Minimum 300 DPI
- [ ] **Color Mode**: RGB (Printful converts to CMYK)
- [ ] **Dimensions**: Match specs above exactly
- [ ] **Bleed**: Add 0.5" bleed if using background colors
- [ ] **Fonts**: Convert to outlines/paths before uploading
- [ ] **Test**: Preview on Printful mockup generator

### Printful Product Recommendations:

**T-Shirts:**
- Bella+Canvas 3001 (Unisex Premium)
- Bella+Canvas 3001CVC (Heather colors)
- Comfort Colors 1717 (Vintage feel)

**Hoodies:**
- Gildan 18500 (Heavy Blend - budget-friendly)
- Next Level 9303 (Premium, softer)

**Mugs:**
- White Glossy Mug 11oz (standard)

## Design Tools

### Free Options:
- **Canva**: Use free version, export as PNG
- **Photopea**: Free Photoshop alternative (photopea.com)
- **GIMP**: Free download, full-featured

### Paid Options:
- **Adobe Illustrator**: Best for vector designs
- **Adobe Photoshop**: Best for photo-based designs
- **Figma**: Great for collaboration

### Quick Start:
1. Use Canva templates sized to specs above
2. Apply brand colors from this guide
3. Download as PNG (transparent background)
4. Upload directly to Printful
