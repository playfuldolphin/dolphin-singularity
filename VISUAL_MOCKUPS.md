## Graphic Style Reference

### Neural Network Pattern:
```
    ●───●
   ╱│╲ ╱│╲
  ● │ ● │ ●
   ╲│╱ ╲│╱
    ●───●
```
- Nodes: White circles
- Connections: Cyan lines (#00c9ff)
- Glow effect: 5px blur, same color

### Dolphin Silhouette Style:
```
       ╱╲
      ╱  ╲___
     │  ◉  ──╲
      ╲____╱  ╲
           ╲__╱
```
- Simple, recognizable shape
- Smooth curves
- Integrated with tech elements

### Sound Wave Pattern:
```
─╱╲─╱╲─╱╲╱╲─╱╲─╱╲─
```
- Varying amplitudes
- Smooth sine curves
- Can be extracted from real dolphin recordings

## Canva Quick Start Templates

### T-Shirt Front (12" x 14"):
1. Create new design: Custom size 3600px x 4200px (at 300 DPI)
2. Add text: Your main headline
3. Apply brand colors from palette
4. Add shapes for graphics
5. Download as PNG (transparent background)

### Hoodie Back (12" x 16"):
1. Create new design: Custom size 3600px x 4800px
2. Layout: Top text → Central graphic → Bottom URL
3. Use grids for alignment
4. Download as PNG (transparent)

### Mug Wrap (8.5" x 3.5"):
1. Create new design: Custom size 2550px x 1050px
2. Keep important elements centered
3. Leave margins on edges
4. Download as PNG (white background or transparent)

## Printful Upload Checklist

For each product:

1. **Choose Product**
   - Navigate to: Products → All Products
   - Select: T-shirt/Hoodie/Mug
   - Choose color and size range

2. **Upload Design**
   - Click "Upload Design"
   - Select your PNG file
   - Position on mockup preview
   - Adjust size to match specs

3. **Set Position**
   - Front: Center chest or as specified
   - Back: Center back or as specified
   - Use ruler guides for precision

4. **Preview**
   - Check all size variations
   - Verify colors look correct
   - Test on different product colors

5. **Set Price**
   - See Printful base cost
   - Add your profit margin
   - Typical markup: 2-3x base cost
   - T-shirts: $25-35 retail
   - Hoodies: $45-65 retail
   - Mugs: $15-25 retail

6. **Publish**
   - Generate product URL
   - Copy URL
   - Add to website (see STORE_SETUP.md)

## Pro Tips

### For Best Results:

1. **High Resolution**: Always use 300 DPI minimum
2. **Color Profiles**: Use RGB for screen, Printful converts
3. **Font Safety**: Convert text to paths/outlines
4. **Test Prints**: Order samples before launching
5. **Mockups**: Use Printful's mockup generator for product photos
6. **Variations**: Offer 3-4 color options per design
7. **Sizing**: Provide size chart on product pages

### Common Mistakes to Avoid:

- ❌ Low resolution images (below 150 DPI)
- ❌ Too much detail (won't print clearly)
- ❌ Text smaller than 0.25" height
- ❌ Colors too similar (low contrast)
- ❌ Designs too close to seams
- ❌ Forgetting bleed area
- ❌ Using CMYK color mode
